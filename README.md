# SecKill

## 系统介绍

本项目是使用SSM框架开发的高并发限时秒杀web应用。



## 开发技术

+ 前端技术 ：Bootstrap + jQuery 

+ 后端技术 ：Spring + SpringMVC + MyBatis + MySQL
+ 数据库： MySQL + Redis
+ 服务器： Tomcat



##开发工具

IntelliJ IDEA + MySQL + Git + Chrome
