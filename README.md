# SecKill

## 项目介绍

本项目是使用SSM框架开发的高并发限时秒杀web应用。

项目功能介绍：

+ 商品秒杀开启前，用户能看到商品秒杀倒计时，但不能进行秒杀。

+ 商品秒杀开启时，可以进行秒杀但不能进行重复秒杀。

+ 商品秒杀结束后，显示商品秒杀已结束，阻止用户进行秒杀。

## 开发技术

+ 前端技术 ：Bootstrap + jQuery 

+ 后端技术 ：Spring + SpringMVC + MyBatis + MySQL
+ 数据库： MySQL + Redis
+ 服务器： Tomcat
